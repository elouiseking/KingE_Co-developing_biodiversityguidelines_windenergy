# KingE_Co-developing_biodiversityguidelines_windenergy
Code and analysis used for Co-developing biodiversity guidelines for the onshore wind energy industry.
This repository contains the R code and data used for the analysis and figures in the paper "Co-developing biodiversity guidelines for the onshore wind energy industry.". The code is contained within two R Quarto document and multiple csv files.

This includes:
Code to generate a bar plot for summary statistics of the analusos within the paper.
Code to analyse interview data.
Code to model the cost-effectiveness anaylsys results.

To run the code in this repository, you need to have R installed. Available at: https://cran.r-project.org/
